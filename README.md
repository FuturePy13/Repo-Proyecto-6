**Análisis de Calidad en Videojuegos: Identificación de Factores Clave y Tendencias del Mercado Gamer**

**Objetivo del proyecto**
Analizar qué factores determinan la calidad percibida de un videojuego utilizando datos reales de calificaciones, géneros, plataformas, características del producto y comportamiento de usuarios, con el fin de obtener insights útiles para decisiones estratégicas de lanzamiento, marketing y desarrollo.

**Tecnologías utilizadas**

* Python

* Pandas

* NumPy

* Matplotlib

* Stats Library / SciPy (resolución de hipótesis y análisis de variación)

**Metodología de trabajo**
1. Limpieza y preparación de datos

* Homologación de nombres de variables

* Manejo de valores faltantes y datos inconsistentes

* Clasificación de juegos por género, plataforma y características principales

* Preparación del dataset para análisis estadístico

2. Análisis Exploratorio (EDA)

* Distribución de valoraciones de usuarios y críticos

* Tendencias de calidad por plataforma (PS, Xbox, Nintendo, PC, etc.)

* Popularidad por género de videojuegos

* Visualización gráfica de variaciones, correlaciones y patrones clave con Matplotlib

3. Análisis de factores asociados a la calidad

* Identificación de variables con mayor impacto en la percepción de calidad

* Comparación de calificaciones entre géneros, plataformas y características específicas

* Relación entre desempeño comercial y valoraciones

4. Validación estadística

Planteamiento y prueba de hipótesis, por ejemplo:
“Los juegos de ciertas plataformas obtienen valoraciones más altas que otros.”

* Análisis de variación para detectar diferencias significativas entre grupos

* Evidencia cuantitativa para soportar recomendaciones de negocio

**Resultados e insights clave**

Algunos géneros, como RPG y Action-Adventure, tienden a recibir calificaciones más altas.

* Las plataformas con bases de usuarios más consolidadas presentan mayor estabilidad en la percepción de calidad.

* Existe una correlación notable entre reseñas de críticos y éxito comercial.

* Las características específicas del juego (duración, complejidad, modo online, etc.) influyen fuertemente en la calidad percibida.
