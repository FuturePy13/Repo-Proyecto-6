
“Repository containing a Data Analysis project with Pandas and included visualizations with Matplotlib, hypothesis resolution and variation values with Stats Library.”

Este proyecto tiene como propósito identificar patrones que determinan la calidad de un videojuego, analizando múltiples variables clave como calificaciones, características del juego, plataformas, géneros y comportamiento de usuarios.
El objetivo final es proporcionar insights accionables que permitan planificar estrategias de lanzamiento, optimizar campañas de marketing y apoyar decisiones de negocio basadas en datos.

Objetivos Principales

* Detectar qué factores influyen más en la percepción de calidad de un juego.

* Identificar tendencias por género, plataforma y características del producto.

* Evaluar la relación entre desempeño comercial y valoraciones de usuarios/críticos.

* Proporcionar recomendaciones para mejorar estrategias futuras de desarrollo y campaña.
